# Installation

## Software requirements

1. Python 3.x.x and R
2. STAR, sambamba, Trinity, RSEM, Bowtie2, GMAP, gffread v0.11.6 and BEDTools (for `SERVE.py`)
3. TACO, StringTie, Cufflinks, GMAP, and gffread (for `SERVE_merge.py`)
4. RSEM and STAR (for `SERVE_quant.py`)

## Obtain SERVE

```bash
git clone https://github.com/janky-yz/SERVE.git
chmod u+x SERVE/script/SERVE*
```
