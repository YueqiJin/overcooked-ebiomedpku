# About SERVE

## About

SERVE is a pipeline for identifying expressed endogenous retroviruses (ERVs).

This repository contains the SERVE pipeline scripts and ERV annotation files:

- Pipeline scripts: https://github.com/janky-yz/SERVE/tree/main/script
- Annotation files: https://github.com/janky-yz/SERVE/tree/main/Annotation
