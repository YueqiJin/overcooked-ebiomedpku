## About

TERA is a pipeline for Transposable element-derived RNA analysis.

The upstream repository includes:

- TERA scripts: https://github.com/janky-yz/TERA/tree/main/script
- TERA reference files (hg19, hg38, chm13v2.0): https://github.com/janky-yz/TERA
